## 🎯 원본을 기반으로 확장한 부분

### AI Native Camp - Day 1 (원본)
정구봉님이 만든 비개발자를 위한 Claude Code 7일 집중 캠프의 첫날 커리큘럼.
- Claude Code 설치 + 7개 핵심 기능 학습
- 슬라이드 없이 Claude가 직접 가르치는 대화형 학습
- `/day1-onboarding` 실행하면 바로 시작

### my-day1의 확장 방향
원본의 **"Claude가 직접 가르치는 대화형 학습"** 철학을 그대로 유지하면서, 비개발자가 더 편안하게 완주할 수 있도록 3가지 방향으로 확장했습니다:

#### 1️⃣ **직무별 맞춤 학습** 🎨
원본의 설명에 더해, 각 직무에 친숙한 비유를 추가했습니다:
- **8가지 직무 프로파일**: HR, 개발자, 디자이너, PM, 분석가, 마케터, 일반 사무, 연구원/학생
- **150개 직무별 비유**: CLI를 HR은 "Excel 수식", 디자이너는 "Figma 단축키"로 설명
- **63개 직무별 동기부여 스토리**: "매번 똑같은 채용 이메일 보내는 데 지치셨나요?" (HR)
- **직무별 실무 과제**: 배운 내용을 자신의 업무에 바로 연결

#### 2️⃣ **Essential Path (점진적 학습)** 🚀
원본의 13개 블록을 유지하되, 학습 부담을 줄이기 위한 선택지를 추가했습니다:
- **필수 7개 경로**: 핵심만 배우고 바로 사용 (1~1.5시간)
- **전체 13개 경로**: 모든 기능 깊이 있게 학습 (2~3시간)
- **커스텀 경로**: 필요한 블록만 선택해서 학습
- **마일스톤 배지**: 🎯 기초 → 🏆 핵심 → 🚀 마스터 (진행 상황 시각화)

#### 3️⃣ **학습 지원 시스템** 🛟
막혔을 때 도움을 받을 수 있는 안전망을 추가했습니다:
- **Help 시스템**: "help" 입력으로 4가지 상황별 도움 (명령어 오류, 용어 이해, skip, 질문)
- **재진입 플로우**: 중단 후 복귀 시 진행 상황 자동 로드
- **용어 비유 시스템**: 기술 용어 출현 시 자동으로 쉬운 비유 제공

## ✨ 주요 특징

### 🎯 Essential Path (v3.0)
- **필수 7개 미션**만 완료해도 실무 투입 가능 (1~1.5시간)
- 심화 6개는 선택적 학습
- 3가지 학습 경로: 빠른 시작 / 전체 / 커스텀

### 💡 동기부여 시스템
- **63개 스토리** (7 필수 미션 × 9 직업군)
- 각 스토리: 공감 질문 → 실제 사례 (Before/After) → 완료 혜택

### 📖 용어 장벽 제거
- **150개 비유** (15 기술 용어 × 10 직업군)
- 직업별 맞춤 설명
  - HR: CLI → "Excel 수식처럼, 텍스트로 컴퓨터에 명령"
  - 디자이너: CLI → "Figma 단축키의 강력 버전"
  - PM: CLI → "프로젝트 커맨드 센터"

### 🏆 마일스톤 배지
- 🎯 **기초 완료**: 미션 0, 1, 3-1 완료
- 🏆 **핵심 완료**: 필수 7개 완료 (실무 투입 가능)
- 🚀 **마스터**: 전체 13개 완료

### ❓ Help 시스템
막혔을 때 "help" 입력 시:
1. 명령어가 안 돼요 → 진단 + 해결법
2. 무슨 말인지 모르겠어요 → 쉬운 재설명
3. 이 미션 건너뛰고 싶어요 → skip 가능
4. 다른 질문이 있어요 → 자유 질문

### 🔄 재진입 플로우
중도 이탈 후 복귀 시:
- 진행 상황 요약 (완료 미션, 진행률, 획득 배지)
- 이어서 하기 / 처음부터 다시 / 특정 미션 선택

## 🚀 사용 방법

### 1. 스킬 실행

```bash
claude
/my-day1
```

### 2. 프로파일 설정 (처음이면)

3가지 질문에 답변:
1. **직업**: HR, 개발자, 디자이너, PM, 분석가, 마케터, 일반 사무, 연구원/학생
2. **전공** (선택): 전공이 있으면 입력
3. **터미널 경험**: 익숙해요 / 해본 적 있어요 / 처음이에요

- **빠른 시작** (필수 7개만, 1~1.5시간) ⭐ 추천

- **커스텀** (필요한 미션만 선택)

### 4. 미션 진행

- 💡 동기부여 스토리 확인
- 📖 용어 출현 시 비유 확인

- ✅ 퀴즈로 이해도 확인
- 🎯 실무 과제로 적용
- 막히면 "help" 입력!
