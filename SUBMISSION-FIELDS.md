# 마켓플레이스 제출 양식 내용

## 📝 Description (플러그인 설명)

```
직업/전공/터미널 경험 기반 개인화 Claude Code 온보딩 스킬입니다.

🎯 Essential Path로 비개발자도 1~1.5시간이면 실무 투입 가능!

주요 기능:
• 필수 7개 / 심화 6개 미션 분류 (완료율 50% → 80% 목표)
• 동기부여 스토리 63개 (7 미션 × 9 직업군)
• 용어 비유 150개 (15 용어 × 10 직업군) - 기술 용어 장벽 제거
• 마일스톤 배지 시스템 (🎯 기초 → 🏆 핵심 → 🚀 마스터)
• Help 시스템 - 막혔을 때 "help" 입력으로 즉시 도움
• 재진입 플로우 - 중단 후에도 이어하기 가능
• 학습 경로 선택 (빠른 시작/전체/커스텀)
• 한국어 완벽 지원 🇰🇷

대상 사용자:
- Claude Code 처음 사용하는 초보자
- 비개발자 (HR, PM, 디자이너, 마케터, 분석가 등)
- 터미널 경험이 없는 현업 실무자
- 빠르게 핵심만 배우고 싶은 사용자

지원 직업군: HR/인사, 백엔드 개발자, UI/UX 디자이너, 제품 기획자/PM, 데이터 분석가, 마케터/콘텐츠 제작자, 일반 사무/관리, 연구원/학생
```

## 📖 Command Content (커맨드 상세 내용)

```markdown
# my-day1: 개인화 Claude Code 온보딩

직업, 전공, 터미널 경험에 따라 맞춤형 예시와 설명을 제공하는 온보딩 스킬입니다.

## 사용 방법

```bash
/my-day1
```

## 특징

### 🎯 Essential Path (v3.0)
필수 7개 미션만 완료해도 실무 투입 가능 (1~1.5시간):
- 미션 0: Setup (5분)
- 미션 1: Experience (10분)
- 미션 3-1: CLAUDE.md (10분) → 🎯 기초 완료 배지
- 미션 3-2: Skill (15분)
- 미션 3-4: Subagent (10분)
- 미션 요약: 7개 기능 요약 (5분)
- 미션 4: Basics (10분) → 🏆 핵심 완료 배지

심화 6개 미션은 선택적 학습 (추가 1시간)

### 💡 동기부여 시스템
각 미션마다 직업별 맞춤 스토리 제공:
- 공감 질문: "매번 똑같은 채용 이메일 보내는 데 지치셨나요?" (HR)
- 실제 사례: Before/After 비교
- 완료 혜택: 3가지 구체적인 이점

총 63개 스토리 (7 필수 미션 × 9 직업군)

### 📖 용어 장벽 제거
기술 용어를 직업별로 쉽게 설명:
- HR: CLI → "Excel 수식처럼, 텍스트로 컴퓨터에 명령하는 방법"
- 디자이너: CLI → "Figma 단축키의 강력 버전"
- PM: CLI → "프로젝트 커맨드 센터"

총 150개 비유 (15 기술 용어 × 10 직업군)

### 🏆 마일스톤 배지
- 🎯 기초 완료: 미션 0, 1, 3-1 완료 시
- 🏆 핵심 완료: 필수 7개 완료 시 (실무 투입 가능)
- 🚀 마스터: 전체 13개 완료 시

### ❓ Help 시스템
막혔을 때 "help" 입력하면:
1. 명령어가 안 돼요 → 진단 + 해결법 제공
2. 무슨 말인지 모르겠어요 → 쉬운 재설명
3. 이 미션 건너뛰고 싶어요 → skip 가능 (필수는 경고)
4. 다른 질문이 있어요 → 자유 질문

### 🔄 재진입 플로우
중도 이탈 후 복귀 시:
- 진행 상황 요약 (완료 미션, 진행률, 획득 배지)
- 3가지 선택: 이어서 하기 / 처음부터 다시 / 특정 미션 선택

## 시작하기

1. **프로파일 설정** (처음 1회)
   - 직업 선택: HR, 개발자, 디자이너, PM, 분석가, 마케터, 일반 사무, 연구원/학생
   - 전공 입력 (선택)
   - 터미널 경험: 익숙해요 / 해본 적 있어요 / 처음이에요

2. **학습 경로 선택**
   - 빠른 시작 (필수 7개만, 1~1.5시간) ⭐ 추천
   - 전체 과정 (13개 전체, 2~3시간)
   - 커스텀 (필요한 미션만 선택)

3. **미션 진행**
   - 💡 동기부여 스토리 확인
   - 📖 용어 출현 시 비유 확인
   - 💻 실습 진행
   - ✅ 퀴즈로 이해도 확인
   - 🎯 실무 과제로 적용
