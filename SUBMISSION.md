### 설명 (Description)
직업/전공/터미널 경험 기반 개인화 Claude Code 온보딩 스킬. Essential Path로 비개발자도 1~1.5시간이면 실무 투입 가능! 동기부여 스토리 63개, 용어 비유 150개, 마일스톤 배지, Help 시스템 포함.

### 짧은 설명 (Short Description, 100자 이내)

### 주요 기능 (Key Features)
1. **Essential Path**: 필수 7개 미션만 완료해도 실무 투입 (1~1.5시간)
2. **동기부여 스토리**: 63개 직업별 맞춤 스토리
3. **용어 비유**: 150개 쉬운 설명 (15 용어 × 10 직업)
4. **마일스톤 배지**: 🎯 기초 / 🏆 핵심 / 🚀 마스터
5. **Help 시스템**: 막혔을 때 즉시 도움
6. **재진입 플로우**: 중단 후에도 이어하기 가능
7. **한국어 완벽 지원**: 비개발자 친화적

### 사용 방법 (Usage)
```bash
/my-day1
```

### 설치 방법 (Installation)
```bash
/plugin install crystal0224/my-day1
```

### 스크린샷 설명
1. 프로파일 설정 화면 (직업/전공/터미널 경험 선택)
2. 학습 경로 선택 (빠른 시작/전체/커스텀)
3. 미션 진행 화면 (동기부여 스토리 + 실습)
4. 마일스톤 배지 획득 화면

- 빠르게 핵심만 배우고 싶은 사용자
